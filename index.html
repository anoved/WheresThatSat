Where’s That Sat?

  • Info
  • Feedback

@WheresThatSat is a Twitter bot that knows where satellites are. Ask it about a satellite, and it will respond with the present coordinates and a link to more detailed information.

Features

Mention a satellite in a tweet to @WheresThatSat, and it will reply with the satellite's coordinates at the time you tweeted. It will also include a link to a map with more information about the satellite's recent path.

If your tweet is associated with a location, the map page will also indicate whether the satellite was potentially visible from that location (not taking into account weather and local variations in terrain).

Alternatively, you can specify an arbitrary location using the special #place hashtag. Name the location in quotes after the hashtag, like this:

@WheresThatSat Envisat #place "Paris, France"

WheresThatSat understands a variety of #place formats. You can give it a street address, the name of a city, or even just a pair of lat/lon coordinates.

Satellite Catalog

When you send a tweet to @WheresThatSat, it scans your message for the names of satellites it recognizes, and will send a reply for each match. It knows about over a thousand satellites! Here is a list of the satellite names it recognizes. Matching is not case sensitive.
