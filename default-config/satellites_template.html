Satellite List

Here is the list of the satellite names recognized by @WheresThatSat. Name recognition is not case-sensitive. The list is periodically synchronized with the set of current two-line element sets available from CelesTrak, and as such may change as additions or deletions are made to the CelesTrak catalog.
